
VW Passat - B1..B9 final exam project
--------------------------------
This package contains static HTML pages for VW Passat B1..B9 generations.
Files:
  - index.html
  - modellek.html
  - b1.html ... b9.html (one page per generation)
  - img/*.png (embedded illustrations)
  - styles.css, script.js
Images are illustrative placeholders generated locally (to ensure offline operation).

How to use:
  1) Unzip the package and open index.html in your browser.
  2) Navigate to 'Modellek' and pick a generation.

If you want, I can later replace the placeholder images with real Wikimedia photos if you provide permission or upload the images.
